Importing Snapshots

Use the Import Snapshots feature to import Debug Snapshot Album (.dsa) files from other projects or users into the Snapshot Albums view for browsing or launching.

Importing a Snapshot Album

  1. In the Snapshot Albums view, click the Import Snapshot ( IMAGE ) icon
  2. Navigate to and select the .dsa file into import
  3. Click Open

NOTE Drag-and-drop support is supported for importing .dsa files.
