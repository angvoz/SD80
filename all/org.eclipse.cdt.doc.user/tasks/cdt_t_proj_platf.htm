Selecting a project type

For Managed Make projects, you need to select the type of project that you plan to create, and the default configurations for your project.

To select a project type:

  1. Select a project type from the list. By default, this dialog dispays only the project types that are buildable on the system on which Eclipse is running. You can view all installed project types by selecting the Show All Project Types button.
  2. Select the configurations that you want added to your project. By default, this dialog dispays only the configurations that are buildable on the system on which Eclipse is running. You can view all default configurations by selecting the Show All Configurations button.
