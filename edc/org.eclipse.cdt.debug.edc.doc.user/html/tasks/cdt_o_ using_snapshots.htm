Using Snapshot Albums

A snapshot album stores one or more individual snapshots for review or sharing. Snapshots are stored in Debugger Snapshot Album (.dsa) files. A snapshot album can be browsed just as you would a photo album, used to launch one or more snapshots, import additional snapshots, or rename snapshots and more.
