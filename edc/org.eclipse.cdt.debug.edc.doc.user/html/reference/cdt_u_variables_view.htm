View Menu

Provides view options as a drop-down menu. Options include:

  • Layout - customize the Variables view layout using these options:
    • Vertical View Orientation
    • Horizontal View Orientation
    • Variables View Only
    • Show Columns - toggles between the column and no column views.
    • Select Columns - opens the Column Select dialog which allows you to set which columns are shown.
  • Number Format - Default, Hex, Decimal, Octal, Binary, and String
  • Show All Variables - toggles between showing all variables and showing only local variables.
