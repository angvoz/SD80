Snapshot Albums View

Use the Snapshot Albums view to browse snapshot albums. Albums are collections of snapshots. Snapshots can be browsed, launched, renamed, deleted, exported and imported as required within this view.

Figure 1 - Snapshot Album view

Snapshot Albums Toolbar

Item Icon Explanation
Refresh
Refresh snapshot information.
Launch Snapshot
Launches the currently selected snapshot in the Carbide debugger. You can also launch a snapshot using a double-click.
Delete Album
Delete the currently selected album or snapshot.
Import Album
Import a snapshot album.
Menu
Provides view options as a drop-down menu.
