Customizing the C-Build view

The C-Build view displays the output of the make utility.

To set C-Build view preferences

  1. Click Window > Preferences.
  2. Expand C/C++, and click Build Console.
  3. To display information on the latest build, select the Always clear C-Build view before building check box.
  4. To open the C-Build view when a project is built, select the Open C-Build view when building check box.
  5. To display the C-Build view if it is already open when a project is built, select the Bring the C-Build view to top when building (if present).
  6. Click OK.
