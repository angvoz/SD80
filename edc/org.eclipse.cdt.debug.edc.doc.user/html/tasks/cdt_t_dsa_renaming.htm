Renaming Snapshots

Snapshots are automatically assigned a name, EXAMPLE, that makes it easy to identify where and when in a debug session it was taken. If this is not sufficient, you can rename snapshots to make them easier for you to locate a specific snapshot. You rename a snapshot using the same method you would for a file.

To Rename a Snapshot

  1. In the Snapshot Albums view, click the snapshot you want to rename
  2. Type the new name, then press the Enter key
