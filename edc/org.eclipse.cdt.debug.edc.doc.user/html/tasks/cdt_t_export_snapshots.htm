Exporting Snapshots

Use the Export Snapshots feature to export Debug Snapshot Album (.dsa) files.

Exporting a Snapshot Album

  1. In the Snapshot Albums view, click the Export Snapshot ( IMAGE ) icon
  2. Navigate to and select the location to save the .dsa file
  3. Click Save
