Make Targets preferences

Use the Make Targets preference panel in the Preferences window to set make target options.

Make Targets Preference Options
Options Description
Build Make target in the background Select this checkbox to perform builds in the background.
Rebuild Last Target This option defines the scope of command "Rebuild Last Target" in Make Targets View.
From selected folder Rebuild the same make target as was run last time in the currently selected folder .
From selected root Rebuild the same make target as was run last time in the project from root folder.
