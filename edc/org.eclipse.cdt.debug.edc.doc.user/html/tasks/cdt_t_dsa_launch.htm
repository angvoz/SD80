Launching Snapshots

You launch a snapshot to reopen the debug session at the exact point of the stored snapshot data. This enables you to view all the data the EDC debugger had captured at the moment the snapshot was taken.

Launching a Snapshot

  • Click the Launch Debug icon () in the Snapshot Albums view toolbar

OR

  • Double-click the snapshot in the Snapshot Albums view

The snapshot is launched, the Debug perspective appears, and the debugger restored to the state saved in the snapshot.
