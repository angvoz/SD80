Customizing the C/C++ editor

You can customize the appearance of the C++ editor by specifying how information is displayed.

To customize the C/C++ editor preferences:

  1. Click Window > Preferences.
  2. Expand C/C++, click C/C++ Editor.
  3. Click the General tab.
  4. Set your preferences and click Apply.
  5. To control the information that is displayed in the margins of the C/C++ editor, click the Annotations tab.
  6. To customize the color of the text on the editable area of the C/C++ editor, click the Colors tab.
  7. Set your preferences and click Apply.
  8. To customize how Code Assist works, click the Code Assist tab. For more information, see

    Using Content Assist

  9. Click OK.
